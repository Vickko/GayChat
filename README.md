### 具体功能

 以下功能均在web端实现

 ​-​ 实现任意用户间的即时通讯和聊天记录加密存储
 ​-​ 支持通讯过程中进行多种文件格式的传递
 ​-​ 支持添加好友功能及好友在线状态查看
 ​-​ 支持群聊功能
 ​-​ 支持在聊天过程中发送表情
 ​-​ 支持对v佬进行光速开盒
